## Big O of common array operations 
### Access
- Space = $O(1)$
- Time = $O(1)$

### Set 
- Space = $O(1)$
- Time = $O(1)$


### Traverse / Search 
- Space = $O(1)$
- Time = $O(n)$

### Copy  
- Space = $O(n)$
- Time = $O(n)$

### Insert
- At beginning 
  - Space = $O(1)$ 
  - Time = $O(n)$
- At the end 
  - Space = $O(1)$
  - Time = $O(1)$
- somewhere in between 
  - Space = $O(1)$
  - Time = $O(n)$
  
### Remove 
- At beginning 
  - Space = $O(1)$ 
  - Time = $O(n)$
- At the end 
  - Space = $O(1)$
  - Time = $O(1)$
- somewhere in between 
  - Space = $O(1)$
  - Time = $O(n)$
